# 途绘 · 旅行规划 H5

## 项目介绍

一款面向手机浏览器的旅行规划应用。用户可以输入目的地和出行天数，由 AI 帮忙生成行程安排；也可以把网上的攻略文字粘贴进来，自动整理成可编辑的行程单，方便查看、修改和分享。

**使用技术：** Vue 3、Vant、NestJS、DeepSeek

---

## 项目功能

| 功能 | 说明 | 状态 |
|------|------|------|
| AI 智能规划 | 输入城市、天数、偏好，生成每日行程 | 开发中 |
| 攻略一键整理 | 粘贴攻略文字，自动提取地点并按天分组 | 开发中 |
| 我的行程 | 查看、编辑、删除已保存的行程 | 开发中 |
| 地图浏览 | 在地图上查看行程路线与地点 | 规划中 |
| 分享行程 | 生成行程长图，便于分享到社交平台 | 规划中 |

---

## 项目规范

### 命名

- 文件夹、文件名：小写 + 连字符，如 `trip-list.vue`
- 变量：小驼峰，如 `tripList`
- 组件：大驼峰，如 `TripCard`

### 前端

- 页面按功能拆分组件，同类 UI 尽量复用
- 页面之间的数据用 Pinia 统一管理
- 接口请求集中在 `api` 目录，不在页面里零散调用

### 后端

- 接口按模块划分：行程、用户、AI 等
- 敏感信息（如 AI 密钥）放在环境变量，不提交到 GitHub
- AI 返回的结果需校验格式，保证前端能正常使用
